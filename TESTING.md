# Claude Plugin Marketplace - 功能测试清单

本文档列出了所有需要手动测试的功能点。

## 前置条件

1. 已安装 Claude Code CLI
2. 已在 VS Code 中加载此扩展（通过 F5 或从本地安装）
3. 已打开一个工作区文件夹

## 测试清单

### 1. 基础功能测试

#### 1.1 扩展激活
- [ ] 打开 VS Code 扩展开发主机
- [ ] 在输出面板查看 `[Claude Plugin Marketplace] Extension is activating...`
- [ ] 确认扩展激活成功，无错误信息
- [ ] 侧边栏显示 "Claude Plugin Marketplace" 视图

#### 1.2 Claude Code 检测
- [ ] 如果 Claude Code 未安装，显示警告消息
- [ ] 警告消息包含 "下载" 按钮
- [ ] 点击 "下载" 按钮打开 Claude.ai 网站
- [ ] 如果 Claude Code 已安装，不显示警告

### 2. 侧边栏视图测试

#### 2.1 树视图显示
- [ ] 展开视图根节点
- [ ] 查看 "已安装插件" 节点
- [ ] 查看市场列表节点

#### 2.2 已安装插件
- [ ] 展开 "已安装插件" 节点
- [ ] 查看已安装的插件列表
- [ ] 每个插件显示正确的名称和版本

#### 2.3 市场列表
- [ ] 展开市场节点
- [ ] 查看市场中的可用插件
- [ ] 每个插件显示名称和描述

### 3. 插件管理测试

#### 3.1 安装插件
- [ ] 右键点击可用插件
- [ ] 选择 "安装"
- [ ] 终端自动打开并执行安装命令
- [ ] 安装完成后显示成功消息
- [ ] 插件出现在 "已安装插件" 列表中

#### 3.2 卸载插件
- [ ] 右键点击已安装插件
- [ ] 选择 "卸载"
- [ ] 显示确认对话框
- [ ] 点击 "确认" 执行卸载
- [ ] 插件从 "已安装插件" 列表中消失

#### 3.3 更新插件
- [ ] 右键点击已安装插件
- [ ] 选择 "更新"
- [ ] 终端执行更新命令
- [ ] 更新完成后显示消息

### 4. 市场管理测试

#### 4.1 添加市场
- [ ] 右键点击根节点
- [ ] 选择 "添加市场"
- [ ] 输入市场名称
- [ ] 输入市场来源（URL/Git/本地路径）
- [ ] 新市场出现在树视图中

#### 4.2 删除市场
- [ ] 右键点击自定义市场
- [ ] 选择 "删除市场"
- [ ] 显示确认对话框
- [ ] 点击 "确认" 删除市场
- [ ] 市场从树视图中消失

#### 4.3 刷新市场
- [ ] 右键点击市场
- [ ] 选择 "刷新市场"
- [ ] 市场中的插件列表更新

### 5. 命令测试

#### 5.1 打开插件市场
- [ ] 按 Ctrl+Shift+P 打开命令面板
- [ ] 输入 "Claude Plugin Marketplace: Open"
- [ ] 执行命令
- [ ] Webview 界面打开

#### 5.2 刷新命令
- [ ] 执行 "Claude Plugin Marketplace: Refresh" 命令
- [ ] 树视图刷新

#### 5.3 搜索插件
- [ ] 执行 "Claude Plugin Marketplace: Search Plugins" 命令
- [ ] 输入搜索词
- [ ] 显示 "搜索功能即将推出" 消息

#### 5.4 查看插件详情
- [ ] 右键点击插件
- [ ] 选择 "查看详情"
- [ ] 显示插件详细信息

### 6. Webview 界面测试

#### 6.1 界面加载
- [ ] Webview 成功打开
- [ ] 界面正确显示
- [ ] 无加载错误

#### 6.2 界面交互
- [ ] 点击插件卡片
- [ ] 查看插件详情
- [ ] 安装/卸载按钮工作正常

### 7. 错误处理测试

#### 7.1 无工作区
- [ ] 关闭所有工作区
- [ ] 重新加载扩展
- [ ] 显示警告消息

#### 7.2 网络错误
- [ ] 断开网络连接
- [ ] 尝试安装插件
- [ ] 显示适当的错误消息

#### 7.3 无效市场
- [ ] 添加无效的市场 URL
- [ ] 显示错误消息
- [ ] 市场不会添加到列表

### 8. 性能测试

#### 8.1 大量插件
- [ ] 有大量插件时，树视图加载流畅
- [ ] 展开节点响应迅速
- [ ] 刷新操作在合理时间内完成

#### 8.2 内存使用
- [ ] 长时间使用后内存占用合理
- [ ] 无明显内存泄漏

### 9. 兼容性测试

#### 9.1 不同 Claude Code 版本
- [ ] 在不同版本的 Claude Code CLI 下测试
- [ ] 确认基本功能正常

#### 9.2 不同操作系统
- [ ] Windows
- [ ] macOS
- [ ] Linux

### 10. 边界情况测试

- [ ] 插件名称包含特殊字符
- [ ] 市场名称为空
- [ ] 快速连续点击多个命令
- [ ] 在安装过程中关闭 VS Code
